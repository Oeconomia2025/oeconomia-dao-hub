# OEC DAO Hub

The governance and analytics dashboard for the Oeconomia DAO ecosystem. Track OEC token metrics, manage your DeFi portfolio, participate in governance, and explore ecosystem protocols — all from one hub.

**Live:** [https://oeconomia.io](https://oeconomia.io)

## Key Features

| Feature        | Description                                                          |
| -------------- | -------------------------------------------------------------------- |
| Dashboard      | OEC token price, market cap, volume, holder stats, recent txs        |
| Analytics      | TVL tracking, volume metrics, protocol filtering, date range charts  |
| Governance     | Proposals, voting (For/Against/Abstain), delegation, quorum tracking |
| Portfolio      | User DeFi positions and holdings across ecosystem protocols          |
| Ecosystem      | Links to all Oeconomia protocols with TVL and APY metrics            |
| Presale        | OEC token presale interface with USDC conversion                     |
